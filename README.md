3. Escreva um programa que retorne o maior número e o menor número encontrado em uma matriz de inteiros.

   | **1** | **5** | **9**  | **13** |

   | **2** | **6** | **10** | **14** |

   ​	

   |         Entrada          |   Saída    |
   | :----------------------: | :--------: |
   | {1, 1, 2, 3, 2, 2, 2, 1} | {2, 2, 2}  |
